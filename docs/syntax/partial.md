偏函数操作符可以让你通过绑定部分参数快速创建一个新的函数。与标准库的 `functools.partial` 相比，它更灵活，更强大。

## 语法

```
f(?, 2)

f(?, ?, 3, ?)

f(a, b=?)

f(*?, b=10)

f(name="Aber", **?)
```

!!! tip ""
    每一个 `?` 都意味着为生成的函数增加一个位置参数。

## 使用技巧

当你需要传递一个序列，但却只想使用管道（`|>`）传递一个参数时，请使用 `*?`。

试试看 `("hello", "world") |> print(?)` 和 `("hello", "world") |> print(*?)` 的区别。
